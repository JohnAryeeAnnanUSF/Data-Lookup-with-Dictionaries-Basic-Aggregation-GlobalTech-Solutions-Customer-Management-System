# GlobalTech Customer Management System

This repository contains a Python-based customer management system developed for a business analytics assignment. The project demonstrates how dictionaries can be used to efficiently organize, manage, and analyze business data.

## Overview

The system simulates how a consulting firm like GlobalTech Solutions tracks customer information, manages project data, and performs financial and operational analysis. It uses Python dictionaries, nested data structures, and aggregation techniques to generate insights from business data.

## Features

* Store and manage customer information using dictionaries
* Organize project data with nested dictionary structures
* Perform customer lookups and handle missing data safely
* Update and modify customer records dynamically
* Calculate project costs based on service rates and hours
* Analyze service usage across multiple projects
* Perform financial aggregations (total hours, budgets, averages)
* Use dictionary methods such as `.keys()`, `.values()`, and `.items()`
* Apply dictionary comprehensions for efficient data processing
* Build functions for validation, budget analysis, and service recommendations
* Generate summary reports for business insights

## Files Included
